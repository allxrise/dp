# DP - Project folder generator with docker's container naming style

- Uses [zig]("https://ziglang.org/") for cross-compilation
- Random names copied/stolen from this [file]("https://github.com/thieman/docker/blob/master/pkg/namesgenerator/names-generator.go")

## Test

- Natively tested on x86_64-linux-gnu
- Used [WINE]("https://www.winehq.org/") for Windows and [Darling]("https://www.darlinghq.org/") for x86_64 MacOS
- "aarch64" equivalent of said platforms will probably work, idk
- Tested NOTHING on BSD platforms, will probably work
